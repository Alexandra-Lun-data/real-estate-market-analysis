# Data Description

## advertisement
- id — listing ID  
- first_day_exposition — listing start date  
- days_exposition — listing duration  
- last_price — property price  

## flats
- id — flat ID  
- city_id — city  
- total_area — area  
- rooms — number of rooms  
- balcony — number of balconies  
- floor — floor  
- ceiling_height — height  
- airports_nearest — distance to airport  
- parks_around3000 — parks nearby  
- ponds_around3000 — ponds nearby  

## city
- city_id — ID  
- city — name  

## type
- type_id — ID  
- type — settlement type
